泰坦

在古希腊和罗马的传说中，天国降血落在大地女神盖雅的膝上，她于是怀孕，产下泰坦族，也就是传说中的巨人族。巨人族身材高大，力大无穷，性情暴躁，令人望而生畏。

 

D级幼年巨人

属性 +2力量 +2耐力

能量池 无

技能树 无

特性 泰坦血脉（正负特性），精通投掷（双特性），灵敏嗅觉，心灵坚韧

特殊：由于没有技能树和能量池，额外获得2个特性

泰坦血脉：泰坦的血脉已经逐渐开始表露，泰坦的意志也开始逐渐影响你的灵魂。你与龙族和其衍生种族成为死敌（仅限西方龙），你面对人类时的交涉-2dp。你的体型+1

精通投掷：你可以将力量作为你投掷的关键属性，你的运动上升一级（不超过4）并且获得专业：重武器投掷

灵敏嗅觉：当你使用嗅觉的相关检定时，获得+3DP。

心灵坚韧：你在遭遇心灵效果影响时，忽略等于你血统等级的点数。

 

 

C级少年巨人

属性 +3力量，+3耐力，+2风度

能量池 无

技能树 无

特性 泰坦血脉（双特性），久战

特殊：由于没有技能树，额外获得1个特性

泰坦血脉 泰坦的血脉已经逐渐开始表露，泰坦的意志也开始逐渐影响你的灵魂。你的体型再次上升2点，同时意志池和意志检定上升2点。

久战：残酷的战争，让你对躲避流矢有了一定的心得，你免疫8点高速优势。

 

 

B级成人礼

属性 +4力量 +4耐力，+1风度，+1决心，+1感知

能量池 无

技能树 无

特性 泰坦血脉（加强），元素之子，猎龙者

特殊：由于没有技能树，额外获得3点属性

泰坦血脉：泰坦的血脉已经逐渐开始表露，泰坦的意志也开始逐渐影响你的灵魂。你的体型再次上升3点。

元素之子：大地会自动抚平你行动的痕迹，当你需要投掷的时候，大地会在你脚下生成大小合适的岩石。花费一个迅捷动作，你的脚边可以生成一块适宜投掷的石块，体积等于你的力量的一半，伤害等于其体积，但是你依然需要花费动作来将它捡起来。

猎龙者：泰坦巨人的成人礼，一般是猎杀成年的巨龙。你在战斗开始时可以进行一次力量+威吓判定，所有可以察觉你的敌人以意志对抗，获得差值的恐惧点数。

 

 

A级独眼巨人

属性 +5力量 +5耐力，+2风度，+2决心，+2感知

能量池 无

技能树 无

特性 泰坦血脉（加强），鹰眼，钢铁肌肤，巨石

特殊：由于没有技能树，额外获得4点属性

泰坦血脉 泰坦的血脉已经逐渐开始表露，泰坦的意志也开始逐渐影响你的灵魂。你的体型值再次上升5点

鹰眼：你觉醒了血脉中的力量，现在你投掷巨石在极限射程之内都不会受到射程减值。并且你可以在模糊视距内一样可以投掷目标的特定部位。

钢铁肌肤：你的皮肤强韧无比，你获得等于你生存技能值的天生防御，并且生存每有1点附加成功，你获得1点全伤害吸收。

巨石：现在你投掷巨石可以造成范围伤害，范围为该石头的体积，伤害为该石头的伤害值，你以力量+运动进行投掷检定，所有范围内的目标反射检定减去你的成功数的DP。

 

 

S级百臂泰坦

属性 +6力量 +6耐力，+6风度，+3决心，+3感知

特性 泰坦血脉（加强）自然化身（双特性）百臂

泰坦血脉：泰坦的血脉已经逐渐开始表露，泰坦的意志也开始逐渐影响你的灵魂，你就是真正的泰坦。你的体型值再次上升5点。

自然化身：你免疫毒素和疾病。你成为了某种自然力量之源的化身你将不在被视为凡物，不接受针对凡物的效果。并且你可以将自己的力量属性的附加成功作为DP额外增加到你所有的检定上。

只要你踩在大地上，你获得5点快速治疗

百臂：你每回合可以额外进行一个标准动作，但是不能用于使用任何需要动作启动的招式，技能或者法术。
